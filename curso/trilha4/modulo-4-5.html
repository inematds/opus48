🧾 A conta = tamanho do job

O custo de um workflow depende de quão grande é o job que você aponta. Cada agente é uma chamada completa do Claude, com seu próprio contexto, e muitos rodam de uma vez. O grosso costuma ser tokens de entrada (mais barato que saída), mas em volume vira muito dinheiro.

⚠️ O caso real do vídeo

Um único prompt vago ("vasculhe meu desktop inteiro") subiu agentes para varrer todos os arquivos e repos — e consumiu cerca de metade de um plano de US$ 200/mês em pouco mais de 30 minutos.

📊 De onde vem o custo

  • •Nº de agentes × contexto de cada um
  • •Modelo escolhido (Opus custa 10–15× mais que Haiku)
  • •Quão amplo é o escopo que você deu
2

🎯 As 3 regras de ouro

As mesmas três regras transformam um workflow caro num barato sem perder qualidade:

🎯
1. Limite o escopo

Diga exatamente onde olhar

📦
2. Nomeie o entregável

"devolva uma tabela"

🪶
3. Haiku no repetitivo

Trabalho braçal barato

✗ Vago → conta alta

"Melhore meu projeto inteiro."

✓ Escopado → centavos

"Cheque cada arquivo em src/routes por auth faltando. 1 agente Haiku por arquivo. Devolva uma tabela."

💾 O script, salvar e reutilizar

O workflow é um arquivo JavaScript que roda em background. Por padrão o run é descartável — mas você pode salvá-lo e transformá-lo num comando.

audit-skills.jsJavaScript
export const meta = { name: "audit-skills" }

const files = listSkills()
await parallel(
  files.map(f => agent("score " + f))   // 1 Haiku por skill
)
// → scorecard ranqueado pior→melhor
O que é→um script JS; um runtime o roda em background enquanto você trabalha.
Salvar→abra /workflows, escolha o run e pressione s.
Reutilizar→vai para .claude/workflows/ e roda como /seu-nome.

💡 Salve no projeto certo

Por padrão o workflow pode ser salvo num diretório global. Se você quer ele dentro do projeto, peça explicitamente para salvar em .claude/workflows/ do projeto. O /deep-research é um workflow embutido que já vem pronto.

5

🚀 UltraCode — máximo esforço no automático

No /effort você tem Low → Max. O UltraCode vai além: é raciocínio X-High + workflows ligados por padrão. Ele orquestra sozinho, sem precisar da palavra "workflow".

  • •Sobe workflows por conta própria — sem palavra-chave
  • •Jobs grandes viram vários workflows em sequência: entender → construir → revisar
  • •Perguntas rápidas e edições pequenas ainda ficam solo

⚠️ O preço da autonomia

O UltraCode para de perguntar e orquestra. Muito mais tokens, muito mais custo. Ligue quando qualidade importa mais que a conta — e desligue para o trabalho do dia a dia.
